# ✂️ Dinu Barber Shop

¡Bienvenido al repositorio de **Dinu Barber Shop**! Este es el sitio web Front-End para una barbería ubicada en Colmenar de Oreja, diseñado para ofrecer una experiencia de usuario rápida, moderna e intuitiva.

El proyecto está desarrollado utilizando **Angular** para la estructura y lógica de la aplicación, y **Tailwind CSS** junto con CSS personalizado para lograr un diseño responsivo y estilizado.

---

## 🚀 Tecnologías Utilizadas

El stack tecnológico principal del proyecto incluye:

*   **Angular** (v18+) - Framework de desarrollo de aplicaciones web.
*   **TypeScript** - Lenguaje de programación principal para la lógica.
*   **Tailwind CSS** - Framework de CSS utilitario para un diseño rápido y moderno.
*   **RxJS** - Para la gestión de flujos de datos asíncronos y eventos.
*   **HTML5 & CSS3** - Para la estructura semántica y estilos específicos.

---
